# 🚀 MTProto Proxy — Универсальный Менеджер v4.4

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Shell Script](https://img.shields.io/badge/Shell_Script-121011?logo=gnu-bash&logoColor=white)

**Простая установка и удобное управление MTProto прокси для Telegram.**  
Скрипт автоматизирует всё: от компиляции до ежедневного мониторинга и интеграции с Telegram-уведомлениями.

> **Важно:** Эта версия (v4.3) фокусируется исключительно на MTProto, предоставляя расширенные возможности мониторинга и управления. SOCKS5 больше не входит в состав скрипта.

---

## ✨ Возможности

*   **🛠 Простая установка:** Автоматическая загрузка зависимостей, компиляция MTProxy и настройка systemd.
*   **📊 Живой мониторинг ресурсов:** Интерактивный дашборд с графиками CPU/RAM, количеством соединений и последними логами, обновляющийся в реальном времени.
*   **🤖 Telegram-интеграция:** Встроенное ядро `tg-core` для отправки статуса прокси и уведомлений в Telegram (с поддержкой нескольких чатов и режимов).
*   **🔧 Полноценный менеджер:** Удобное меню для запуска, остановки, смены порта, применения спонсорского тега и просмотра логов.
*   **🔒 Безопасность:** Генерация надежного секрета, возможность запуска от непривилегированного пользователя `mtproxy` с использованием `capabilities`.

## 📦 Установка

### Требования
*   **ОС:** Debian, Ubuntu, CentOS (и другие systemd-дистрибутивы)
*   **Права:** Root (`sudo`)
*   **Интернет:** Для скачивания зависимостей и кода

### Быстрая установка (curl)

```bash
curl -fsSL https://raw.githubusercontent.com/tarpy-socdev/MTP-manager/refs/heads/deving/mtproto-universal-v4.sh -o mtproto-universal-v4.sh
chmod +x mtproto-universal-v4.sh
sudo ./mtproto-universal-v4.sh
